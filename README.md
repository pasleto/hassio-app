Desktop application for **Home Assistant** available for Windows and MacOS.

Features:
  - Local URL
  - Remote URL
  - Tray Icon
  - Start at login
  - Start hidden
  - Start maximized

Cons:
  - No AutoUpdate the app
    - needs Apple Developer License and Microsoft Developer License in order to work
